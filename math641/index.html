Announcements

  • There is a new typsetting system, Typst, that aims to eventually rival LaTeX. It’s fast and has a clean syntax with modern programming capabilities. Although it is still immature (its public beta started last winter), I’m pretty impressed with its potential. I’ll be using it myself for day-to-day typesetting this semester. You might want to use it for your solutions, either via the web app (with instant preview!), or by installing the compiler). You’ll want the course template to get started. In case you are curious:
